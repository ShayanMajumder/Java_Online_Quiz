## Requirements

`Java 17`
## Installation

```bash
$ git clone https://github.com/ShayanMajumder/Java_Online_Quiz.git
$ cd Java_Online_Quiz
```



### Student Quiz attempt
Attempt the student quiz by following the given steps
```bash
$ javac registration.java
$ java registration
```
### Admin Panel
Attempt the student quiz by following the given steps
```bash
$ javac admin.java
$ java admin
```
